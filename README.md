# CSC-2022-Project
## Simple ATM (Python + Tkinter)
A basic ATM simulation with GUI.
## Features:
- Check Balance 
- Deposit Money
- Withdraw Money
- Error handling for invalid input
## Tech Used:
Python, Tkinter
## How to Run:
```bash
python atm.py
```
##  Student Marklist System (C++)
A console-based program to manage student marks and calculate results.
## Features:
- Add multiple students
- Enter subject marks
- Calculate total
- Calculate percentage
- Display formatted mark list
## Tech Used:
C++, OOP Concepts, STL Vector
## How to Run:
```bash
g++ mark_system.cpp -o mark
./mark
```
